# Crowd Counting using Machine Learning

##  Project Overview

This project estimates the number of people in an image using computer vision and machine learning techniques.

##  Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn (Random Forest)
* Google Colab

##  Approach

* Extracted features like brightness, variance, edge density, and texture
* Used Random Forest Regressor to predict crowd count
* Evaluated using MAE, MSE, and RMSE

## Output

* Predicted vs actual comparison
* Error distribution analysis
* Feature importance visualization
